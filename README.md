# Sofia's English — Unit 5

Web app per ripassare Unit 5 ("Bath, here we come!") del libro *Hello World - Classe 5* (Celtic Publishing / Raffaello).

## Funzionalità
- **Flashcards** con pronuncia inglese (Web Speech API)
- **Quiz** a scelta multipla, 10 domande estratte da pool di 20
- **Matching** parola↔immagine, 6 coppie casuali
- **Fill the gap** completa la frase
- Progressi salvati in `localStorage`
- Installabile come PWA sulla home del telefono (manifest + service worker)

## Sviluppo
Sito statico: apri `index.html` o servi con `python -m http.server`.

## Deploy
GitHub Pages — basta `main` branch, root.

## Contenuti
Tutti i topic di Unit 5: Town places, Directions & Prepositions, Imperatives, Romans in Britain, Roman Baths.
